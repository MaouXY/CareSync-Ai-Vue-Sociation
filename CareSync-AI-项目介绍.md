# CareSync AI 项目介绍

## 项目概述

CareSync AI 是一个专注于留守儿童情感陪伴与服务的创新平台，致力于通过"AI 智能陪伴 + 社工精准介入"的模式，为留守儿童提供有温度的情感支持，同时为社工提供数据驱动的精准服务工具。

### 核心价值
- **情感陪伴**：利用AI技术为留守儿童提供24小时智能情感支持
- **精准服务**：为社工提供数据分析工具，实现精准干预和服务
- **数据驱动**：基于数据分析提供科学的决策支持
- **温暖关怀**：在技术驱动下保持人文关怀的温度

## 项目背景

留守儿童是中国社会发展中的一个重要群体，他们面临着情感缺失、心理发展、教育资源不足等多重挑战。CareSync AI 通过现代信息技术手段，为这一群体提供更专业、更温暖的服务支持。

## 核心功能

### 1. AI智能陪伴系统
- **24小时情感支持**：AI助手全天候陪伴，及时响应儿童情感需求
- **个性化交流**：基于儿童特点和需求，提供个性化的交流体验
- **情感识别**：通过对话识别儿童情绪状态，提供针对性关怀
- **知识辅导**：在学习、生活等方面提供智能辅导

### 2. 社工服务管理
- **儿童档案管理**：建立完整的儿童信息档案，跟踪成长轨迹
- **风险评估**：基于多维度数据分析，识别高风险儿童
- **服务方案制定**：AI辅助制定个性化的服务干预方案
- **进度跟踪**：实时跟踪服务效果，调整服务策略

### 3. 数据分析平台
- **实时监控**：儿童状态、服务效果的实时监控
- **趋势分析**：基于历史数据的趋势分析和预测
- **效果评估**：服务效果的量化评估和可视化展示
- **决策支持**：为管理决策提供数据依据

### 4. 智能预警系统
- **风险预警**：及时发现潜在风险，主动介入干预
- **异常监测**：监测儿童行为异常，及时跟进处理
- **服务质量监控**：确保服务质量和效果

## 技术架构

### 前端技术栈
- **Vue 3 + TypeScript**：现代化前端框架，提供类型安全和组件化开发
- **Vite**：快速的构建工具，优化开发体验
- **Arco Design Vue**：企业级UI组件库，保证界面一致性
- **Pinia**：轻量级状态管理，高效的数据流管理
- **Vue Router**：单页应用路由管理
- **ECharts**：数据可视化图表库

### 开发工具链
- **npm**：包管理器，统一依赖管理
- **ESLint + Prettier**：代码规范和质量保证
- **Git**：版本控制，团队协作开发
- **VS Code**：推荐IDE，丰富的插件生态

## 开发环境搭建

### 环境要求
- Node.js >= 16.x
- npm >= 8.x
- Git

### 安装步骤

1. **克隆项目**
```bash
git clone [项目地址]
cd CareSync-Ai-Vue-Sociation
```

2. **安装依赖**
```bash
npm install
```

3. **启动开发服务器**
```bash
npm run dev
```

4. **构建生产版本**
```bash
npm run build
```

### 开发命令
```bash
npm run dev          # 启动开发服务器
npm run build        # 构建生产版本
npm run preview      # 预览生产版本
npm run lint         # 代码规范检查
npm run format       # 代码格式化
```
